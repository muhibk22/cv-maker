# CV-Maker
A responsive, customizable CV/Resume builder built with React. This project allows users to input their personal, educational, and professional information through dynamic forms and instantly preview a formatted resume. Features include live preview, print-ready layout, support for custom fonts, section toggling, and the ability to export as PDF.
